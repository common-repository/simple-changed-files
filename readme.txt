A very simple plugin that displays the names of files changed after a given time or within a time range.

== Description ==

This plugin allows administrators to view the names of code files changed after a given time or within a time range.
This can be useful for:

* detecting changes made by malware
* change management for developers and project managers
* running a preliminary scan before using a file monitoring system like "WordPress File Monitor Plus"

== Installation ==

1. Use the Plugins, Add New menu in WordPress to install the plugin or upload the `simple-changed-files` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the "Simple Changed Files" menu item in the Settings menu to get a list of files
